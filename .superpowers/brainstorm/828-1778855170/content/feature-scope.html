功能范围

Everything HTTP 是只读的 — 我们在此之上能做哪些操作？

核心功能（必选）

🔍
文件搜索

支持 Everything 完整搜索语法（布尔运算、通配符、正则、按扩展名/大小/日期过滤）

📂
目录浏览

点击文件夹进入，面包屑导航，路径显示

⬇️
文件下载

通过代理流式下载，保留原文件名，支持大文件

可选增强（需要你的确认）

1

文件预览

图片缩略图 + 文本文件在线预览。需要 API 代理获取文件内容

2

收藏夹 / 快速访问

固定常用目录，localStorage 持久化，一键跳转

3

目录树导航

左侧可展开的文件夹树，按需加载子目录（调用 Everything API 逐层获取）

这个是多选，你可以选择多个增强功能。

Everything 连接配置

工具需要知道 Everything HTTP 服务器的地址，你倾向哪种配置方式？

A

环境变量 / 配置文件

在 .env.local 或 settings 中配置 Everything HTTP 地址和端口，服务端读取

B

页面内配置

首次使用时在 UI 中输入 Everything HTTP 地址，localStorage 持久化，API 请求时传给服务端

C

自动检测 + 手动备选

先尝试默认端口 localhost:8080，不可用则提示用户输入配置
